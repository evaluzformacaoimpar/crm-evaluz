Configuração do CRM

Siga os passos abaixo para ativar o seu CRM. Tempo estimado: 20 minutos.

1
Criar conta no Supabase (banco de dados gratuito)
  1. Acesse supabase.com e clique em "Start your project"
  2. Clique em "Sign up with GitHub" ou crie uma conta com o email evamentorfuturo@gmail.com
  3. Confirme o email se necessário
  4. Após o login, clique em "New Project"
  5. Preencha: Organization = Eva Luz Vendas, Project Name = crm-evaluz
  6. Escolha uma senha forte para o banco (anote em local seguro)
  7. Região: South America (São Paulo)
  8. Clique em "Create new project" e aguarde 2 minutos
Guarde a senha do banco que você definiu. Você vai precisar dela se precisar acessar o banco diretamente.
2
Copiar as credenciais do Supabase
  1. No painel do Supabase, clique em Settings (ícone de engrenagem) no menu lateral
  2. Clique em API
  3. Copie o valor de Project URL (começa com https://...supabase.co)
  4. Copie o valor de anon (public) na seção "Project API keys"
Você vai usar esses dois valores no Passo 5.
3
Executar o schema (criar as tabelas do banco)
  1. No Supabase, clique em SQL Editor no menu lateral (ícone de banco de dados)
  2. Clique em "New query"
  3. Abra o arquivo schema.sql que está na pasta do CRM com o Bloco de Notas
  4. Selecione todo o conteúdo (Ctrl+A) e copie (Ctrl+C)
  5. Cole no SQL Editor do Supabase
  6. Clique em Run (ou pressione Ctrl+Enter)
  7. Deve aparecer a mensagem "Success"
Se aparecer algum erro, verifique se copiou o conteúdo completo do arquivo schema.sql.
4
Criar sua conta de administradora
  1. No Supabase, clique em Authentication no menu lateral
  2. Clique em Users
  3. Clique em "Invite user"
  4. Digite seu email: evamentorfuturo@gmail.com
  5. Clique em "Invite"
  6. Verifique seu Gmail: chegará um email de convite. Clique no link para criar sua senha
  7. Anote o User ID que aparece na lista de usuários (uma sequência de letras e números, tipo: abc123-def-...)
5
Inserir seu perfil de administradora no banco
  1. No Supabase, clique em Table Editor
  2. Clique na tabela profiles
  3. Clique em "Insert row"
  4. Preencha:
    • id = cole o User ID copiado no passo anterior
    • name = Eva Luz
    • role = admin
  5. Clique em Save
6
Preencher as credenciais no arquivo config.js
  1. Abra a pasta CRM-EVALUZ
  2. Abra a pasta js
  3. Clique com o botão direito em config.js e escolha "Abrir com Bloco de Notas"
  4. Substitua COLE_AQUI_SUA_URL_DO_SUPABASE pela URL copiada no Passo 2
  5. Substitua COLE_AQUI_SUA_ANON_KEY_DO_SUPABASE pela anon key copiada no Passo 2
  6. Salve o arquivo (Ctrl+S)
const SUPABASE_URL = 'https://SEU-PROJETO.supabase.co'; const SUPABASE_ANON_KEY = 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...';
7
Criar conta no Netlify (hospedagem gratuita)
  1. Acesse netlify.com
  2. Clique em "Sign up" e escolha "Sign up with email"
  3. Use o email evamentorfuturo@gmail.com
  4. Confirme o email
8
Publicar o CRM no Netlify (arrastar a pasta)
  1. No Netlify, na tela inicial, você verá uma área que diz "drag and drop your site folder here"
  2. Abra o Windows Explorer e navegue até C:\Users\Eva\
  3. Localize a pasta CRM-EVALUZ
  4. Arraste a pasta inteira para a área indicada no Netlify
  5. Aguarde o upload (pode levar 1-2 minutos)
  6. O Netlify vai gerar um endereço como https://nome-aleatorio.netlify.app
  7. Acesse esse endereço no celular e no computador. O CRM vai funcionar nos dois!
Anote o endereço gerado. Você pode personalizar para algo como crm-evaluz.netlify.app nas configurações do site.
9
Configurar lembretes por email Opcional
  1. Acesse emailjs.com e crie uma conta gratuita
  2. Em Email Services, conecte sua conta Gmail
  3. Em Email Templates, crie um template com as variáveis:
    {{to_name}}, {{contact_name}}, {{task_title}}, {{due_date}}
  4. Copie o Service ID, Template ID e Public Key
  5. Cole cada valor no arquivo js/config.js nos campos correspondentes
  6. Refaça o upload da pasta no Netlify (arraste novamente)
✓
Pronto! Seu CRM está no ar.

Acesse o endereço gerado pelo Netlify no seu computador e no celular.

Login com: evamentorfuturo@gmail.com e a senha que você criou.

Para convidar colaboradores: acesse o menu Usuários dentro do CRM.
